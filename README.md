# 🎞 Blockmaster

A Movies & TV Series discovery platform that lets users search for films (latest releases or classic titles) and view detailed information.

## 🔗 Live Demo

Visit Blockmaster: **[blockmasterf7.vercel.app](https://blockmasterf7.vercel.app)**

---

## 📋 Prerequisites

- **Node.js** ≥ 18

---

## 🚀 Quick Start

```bash
# Clone the repository
git clone <repository-url>
cd blockmaster

# Install dependencies
npm install

# Start development server
npm run dev
```

---

## 📦 Available Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start the development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview the production build |
| `npm run lint` | Run ESLint |

---

## 🛠 Technologies

- **React** — UI framework
- **Vite** — Build tool
- **React Router** — Client-side routing
- **Bootstrap** — UI components
- **Styled Components** — CSS-in-JS
- **Zustand** — State management
- **ESLint** — Code linting
- **TypeScript** — Type safety

---

## ✨ Features

- **Search movies and TV series** — Find titles by name
- **View details** — See release date, rating, and overview
- **Favorites** — Save movies to your favorites list

---

## 🏗 Building & Deployment

```bash
# Create a production build
npm run build
```

The build output is in the `dist/` folder. Deploy this folder to any static hosting service (Vercel, Netlify, GitHub Pages, etc.).
